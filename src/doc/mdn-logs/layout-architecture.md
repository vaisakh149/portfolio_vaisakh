# Layout Architecture

## Why We Use Layouts

Layouts help maintain a reusable structure across pages.

Instead of repeating navigation, sidebar,
and wrappers on every page, we create a centralized layout.

---

## In This Project

We use:

- MainLayout
- Sidebar
- Topbar

This helps:
- responsiveness
- maintainability
- scalability

---

## Desktop Layout

Desktop uses:
- sticky sidebar
- content workspace

---

## Mobile Layout

Mobile uses:
- top navigation
- collapsible menu

---

## Benefits

- reusable structure
- easier responsive design
- cleaner pages
- modular architecture
